02a - HTML

Fachbegriff Erklärung
Audio Das "audio"-Tag ermöglicht die Einbindung von Audio-Dateien in die Website. Über eine Kontrollleiste kann der Anwender die Wiedergabe steuern, aber auch das automatische Abspielen beim Aufruf der Seite ist möglich. Wie bei Videos können auch hier mehrere Audioquellen angegeben werden. Der Browser wählt dann ein kompatibles Format selbstständig aus.
Anker/Hyperlink Das Anker-Element dient der Verlinkung innerhalb einer Seite oder auf andere Webseiten anderer oder derselben Domain.
HTML The HyperText Markup Language, or HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.
DOM Das Document Object Model (DOM, engl. für Dokumenten-Objekt-Modell) ist eine Spezifikation einer Programmierschnittstelle, welche HTML- oder XML-Dokumente als eine Baumstruktur darstellt, in der jeder Knoten ein Objekt ist, welches einen Teil des Dokumentes repräsentiert, z. B. einen Absatz, eine Überschrift, ein Video oder etwa eine Tabellenzelle. Die Schnittstelle ist plattform- und programmiersprachenunabhängig und erlaubt damit standardisiert, die Struktur und das Layout eines Dokumentes zu verändern. Im Webbrowser bildet dies einen wichtigen Baustein für dynamische Webseiten.
Listen Listen sind Aufstellungen von Informationen in einheitlicher Form und fortlaufender Folge. In HTML gibt es 3 grundsätzlich verschiedene Listentypen, die neben dem Fließtext zu den häufigsten Textstrukturierungs-Elementen gehören: Beschreibungslisten (<dl>), Listen mit fester Reihenfolge (<ol>) und Aufzählungslisten ohne Reihenfolge (<ul>). Einsatzzweck sind unter anderem Navigationen, Auflistungen und Adressfelder.
Tabellen Die HTML-Tabellen ermöglichen Web-Autoren Daten, wie Texte, Bilder, Links, andere Tabellen etc., in Zeilen und Spalten von Zellen zu arrangieren. Die HTML-Tabellen werden unter Verwendung des "table"-Tags erstellt. Eine Tabelle unterteilt sich dann in "thead" und "tbody". Mithilfe des "tr"-Tags werden Tabellenzeilen erstellt. Das "td"-Tag wird verwendet, um Datenzellen zu erzeugen.
Videos Das "video"-Element wird verwendet, um Videos einzubetten. Es kann verschiedene Quellen für Videos enthalten, die im src-Attribut oder im "source"-Element (en-US) repräsentiert werden. Der Webbrowser wählt dann die für ihn passende Quelle aus.
Canvas Das "canvas"-Element wird verwendet, um Grafiken im laufenden Betrieb über JavaScript zu zeichnen. Das "canvas"-Element ist nur ein Container für Grafiken. Sie müssen JavaScript verwenden, um die Grafiken tatsächlich zu zeichnen. Canvas bietet verschiedene Methoden zum Zeichnen von Pfaden, Feldern, Kreisen, Text und zum Hinzufügen von Bildern.
