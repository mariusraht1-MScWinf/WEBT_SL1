02b - CSS

Fachbegriff Erklärung
CSS Das CSS (Abkürzung für Cascading Style Sheets) in der Informatik ist eine Sprache, mit der die Formatierung von HTML- , XHTML- und XML- Dokumenten wie Websites und verwandten Webseiten definiert wird . Die Regeln für die Erstellung des CSS sind in einer Reihe von Richtlinien (Empfehlungen) enthalten, die seit 1996 vom W3C herausgegeben wurden .
Attribute in CSS Es ist möglich, HTML-Elemente mit bestimmten Attributen oder Attributwerten zu formatieren. Der [attribute]-Selektor wird verwendet, um Elemente mit einem bestimmten Attribut auszuwählen.
Klassen in CSS Eine Klasse ist in CSS - einfach gesagt - eine Gruppierung von Eigenschaften, die einmal definiert wird und in der Regel mehrmals verwendet wird. Klassen können in CSS mehrfach angewendet werden. IDs dagegen können aufgrund der Eigenschaft immer eindeutig identifizierbar zu sein, nur einmal pro Seite verwendet werden.
ID-Selektor in CSS Ein CSS-ID-Selektor verwendet das ID-Attribut eines HTML-Elements, um ein eindeutiges Element auf einer Seite auszuwählen. Um einen ID-Selektor in CSS zu verwenden, schreiben Sie einfach ein Hashtag (#) gefolgt von der ID des Elements. Fügen Sie dann die Stileigenschaften, die Sie auf das Element anwenden möchten, in Klammern ein.
Pseudoklassen Bei Pseudoklassen handelt es sich um einfache Selektoren, die ein Element dann ansprechen, wenn es eine bestimmte Eigenschaft besitzt. So lassen sich zum Beispiel Elemente auswählen, über denen sich gerade der Mauszeiger befindet oder Elemente, die das erste Kindelement eines anderen Elementes sind.
list-style-type Die CSS-Eigenschaft "list-style-type" bestimmt, wie die Einträge einer Liste ausgezeichnet werden. Übliche Arten sind etwa Punkte oder eine Nummerierung vor jedem Eintrag. Meist wird diese Eigenschaft mit dem "li"-Element in Verbindung gebracht. Tatsächlich kann es sich jedoch um jedes beliebige Element handeln, dessen display-Eigenschaft list-item ist.
Style Sheet Stylesheets stellen einen großen Durchbruch für Webseiten-Designer dar und erweitern ihre Fähigkeit, das Erscheinungsbild ihrer Seiten zu verbessern. Ein Stylesheet ist eine Datei oder ein Formular, das in der Textverarbeitung und im Desktop Publishing verwendet wird, um den Layoutstil eines Dokuments zu definieren. Ein Stylesheet enthält die Spezifikationen des Layouts eines Dokuments, z. B. Seitengröße, Ränder, Schriftarten und Schriftgrößen. In modernen Textverarbeitungsprogrammen wie Microsoft Word wird ein Stylesheet als Vorlage bezeichnet. Die bekannteste Form des Stylesheets ist das Cascading Style Sheet (CSS), mit dem Webseiten gestaltet werden.
